Basic Installation

To get this template running on your own server upload the contents of the folder to the root of your webserver.

Take a look at the included page templates to get an idea of what is possible and how you'd like to configure your website. The instructions below will guide you through setting up your own preferred layout.

Responsive Design

Bootstrap is providing some of the responsive structure for this design, mainly the fluid collumns. The bottom of the style.css file holds additional modications to make this interface function the way it does.

The footer area of this template uses a lot of custom CSS for it's responsive structure. Each page of this template is responsive and will adjust to fit desktops, laptops, iPhones, iPads, Android phones and tablets and more.

If you are new to responsive design there are two important things to remember when making adjustments to this theme.

  1. Be careful when modifying a style, to make sure that you check if it is appearing later in the document under a media query, as you may need to modify it there as well. Forgetting to do this can result in unexpected breaking of your designs layout when activating media queries. It is always easier to do this as you go rather than trying to remember each modififcation later.
  2. There are window based media queries and device width based media queries. Some media queries target a specific device only, such as the media query for an iPad in portrait position with device widths. When you resize your browser window this media query will not be activated because it only applies to a device with matching screen dimensions. To test modifications to this in a browser you can temporarily change the media query from "min-device-width" to "min-width".

Important Classes

When searching for a part of the template that you would like to edit or implement, remember firebug and the web inspector can help you find the ids and classes you need quickly. If you are not familiar with Bootstrap's layout scaffolding, take a look here first: http://twitter.github.com/bootstrap/scaffolding.html#layouts

Here are a few custom classes you will probably be working with often and what they do:

  • .border-row: Applies a border, shadow and light background to a container. Can be applied to any container with that already has a span class or row-fluid class. This styling is present on many areas of the template. An example is the content of the FAQ page. Its light background and border box come from the .border-row class.
  • .dark-row: Applies a dark transparent gradient to a container. Can be applied to any container with that already has a span class or row-fluid class. You can find an example of this class in action at the icon feature area under the testimonial slider in index.html
  • .standout: Applied to a row-fluid container to create a block of text that catches the eye. See the example text directly under the slider on home2.html.
  • .img-circle: uses border-radius to create a circle image. Apply to any img tag as a class.
  • .home-action: Creates a contrasting call to action block. Apply to any span class to set it's width. The "get started today" text just before the footer on home2.html uses this style.
  • .padding-left: when creating a multi-column layout inside a .border-row class container as described above, you may wish to add some padding around your content. Apply this class to add left spacing.
  • .padding-right: reverse of padding-left. Apply to containers with the class border-row.
  • .padding-both: Same as above, but applies padding to both sides.
  • .sidebar-section: creates a container in your sidebar with padding and a dividing border below. See any template with a sidebar for an example.
  • .blog-box: creates a styled container for an individual blog post. Can be used for post previews and for entire posts.

Buttons

This theme includes a variety of custom designed CSS3 buttons. Implementing them is easy. Simply create an anchor element with the class "btn". 

<a href="#" class="default-btn">Normal</a>

Adding the class .small-btn .large-btn .xlarge-btn or .xxl-btn will change the button size.

Adding the color class next to the button will change the button color. For example: .default-btn is a standard blue button. Adding the .pink-btn class in addition will make the button turn pink.

To find all the button classes in style.css, search for @BUTTONS.

Note: Bootstrap's button styles are preserved and still available for use as well. See buttons.html for a display of all button options.

Retina Web Font Icons

This theme includes multiple sets of web font icons. These icons are compatible with all browsers down to IE7. The main set includes all of the icons you would typically find in Twitter's Bootstrap, converted into web font icons. They are implemented the same way you would implement an icon in Boostrap and all the class names are the same.

The Font Awesome Icons use a Creative Commons license and the author requires that you leave the comment above the CSS like this: 

<!-- Retina Webfont for Boostrap from Font Awesome - http://fortawesome.github.com/Font-Awesome -->

Besides looking great on retina displays, webfont icons can have their color set in CSS like any other font. You can also apply effects like text-shadows and transitions.

Also included is a social icon font set called WLM (see link below for source). WLM social icons are used for the icons in the footer of this template.

Icon source files are located in the "/fonts" folder. Learn more about each icon set at the links below.

  • http://fortawesome.github.com/Font-Awesome/
  • http://www.fontspace.com/wlm-fonts/wlm-web-iconized

Javascript Sliders

The sliders used in this theme are carousels from Bootstrap with a couple additions. To achieve the same style as the main slider on index.html you'll need to give your slider an id of #myCarousel. To create a slider like the testimonial slider on testimonials.html you will need to give your slider an id of #TestimonialSlide.

The #myCarousel homepage slider can be set to multiple widths as shown on home2.html where it fills half of the page width. Each slide can include a caption and an additional large text option has been added.

To see an example of how to add the large text over an image look at line 188 in index.html. Below is a code sample for large captions.

<div class="slider-hero-text hidden-phone">
  <h2>A Responsive Retina Ready Theme</h2>
  <p>36 Page Templates Included!</p>
</div>

Bootstrap's sliders are responsive, and both of these custom styles work great at any viewport size. Learn more about these sliders from Bootstrap's documentation: http://twitter.github.com/bootstrap/javascript.html#carousel

Note: The large text captions shown above are hidden on smartphones so the slider image is not covered. To include captions on smartphones use the regular image captions. See index.html starting at line 183 for an example.

Lightbox and Portfolio Setup

This theme includes 5 portfolio page templates. Image captions for these pages have 4 optional animated CSS3 hover effects. Clicking on any of these images triggers a lightbox effect to view a larger image.

Setting Up your Portfolio

A portfolio image can be as small as a .span3 column or as large as a .span12 single column. Images widths can be mixed as seen on the gallery.html template. The only adjustment needed when creating an image as wide as a .span6 column or wider is that the ".wide" class should be appended after the .portfolio-box class to center the image properly when it is made narrow.

The figure tag (lines 3-10 in the code below) contains the portfolio image and its caption.

To adjust the animation effect all you need to do is change the class after .portfolio-box on line 3. The code below is set to the .slide-top animation. There are four different animation classes. Experiment with them and choose the one you like or even alternate them.

  • .slide-top
  • .slide-right
  • .fall-in
  • .fade-in

Set your image on line 5. Then change the h4 and p contents in figcaption to set your title and description for each portfolio item. This is shown in lines 7 and 8 below.

  <figure class="portfolio-box slide-top">
    <i class="icon-info-sign"></i> 
      <img src="images/port1.jpg" alt="" />
      <figcaption class="portfolio-details">
          <h4>Portfolio Item Header</h4>
          <p>Description of Portfolio Item. How it was made. Who it was made for.</p>
      </figcaption>                     
  </figure>
  </a>
  <div class="lightbox fade" id="box1" style="display: none;">
    <div class='lightbox-content'>
      <img src="images/port1.jpg" alt="">
    </div>
  </div>
</div><!-- End portfolio item span column -->

Google Maps

To adjust Google Maps to your business' location you will need to change the lat and long coordinates in two lines of code. Both lines are shown below. Open contact.html and the first line can be found on at line 30. The second line shown is line 108. The first line sets the map center and the second sets the map marker position.

var monterey = new google.maps.LatLng(39.95227, -75.1636);
var markerPos = new google.maps.LatLng(39.95227, -75.1636);

Find the general documentation on the Google Maps API here: https://developers.google.com/maps/documentation/javascript/. If you have an adwords account you can configure this map to display ads. See this page for more info: https://developers.google.com/maps/documentation/javascript/advertising

This theme uses a customized theme for google maps. Visiting http://gmaps-samples-v3.googlecode.com/svn/trunk/styledmaps/wizard/index.html will allow you to customize your own map colors.

Mailchimp Subscribe Form

The footer of this template has a AJAX email subscribe form with validation. Learn how to configure it for your mail list below. If you do not have a mailchimp account you can signup for free at http://mailchimp.com.

  • Create a new email list and an email form.
  • Give the form a name field and an email field. Save the form.
  • In the form builder you will see a tab near the top that says "Share It". Click this tab.
  • Next click the "Create HTML code for a small subscription form" button on the right side.
  • Now scroll to the code section at the bottom right and we will need to copy a few small parts to replace in the index.html file of your theme.

Copy the code near the top in between the quotes of <form action="">. It will look something like this, but you only need to copy what is in the quotes:

<form action="http://velorastudios.us5.list-manage.com/subscribe/post?u=3eb59c0744a850c1b1cab4c95&id=a65e7f5a53">

Paste that code in between the quotes of form action="" in the index.html file of the template replacing the old action. Next scroll down to the javascript in the code (about halfway down) on the mailchimp embed code and copy the URL in between the quotes after you see options = { url: ''}

It will look like this:

options = { url: 'http://velorastudios.us5.list-manage2.com/subscribe/post-json?u=3eb59c0744a850c1b1cab4c95&id=a65e7f5a53&c=?',

Paste that url in the same area in between the quotes on the index.html file. Save the index.html file of the theme.

Try out the form to make sure it works and you are all ready to go!
